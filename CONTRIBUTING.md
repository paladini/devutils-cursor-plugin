# Contributing

Thanks for helping improve DevUtils!

## Where to contribute

| Change | Repository |
| --- | --- |
| New tools, server bugs, Docker, npm | [devutils-mcp-server](https://github.com/paladini/devutils-mcp-server) |
| Plugin manifest, Cursor/Claude install, docs | [devutils-cursor-plugin](https://github.com/paladini/devutils-cursor-plugin) (this repo) |

## Plugin changes

1. Fork and clone this repo
2. Edit manifests (`.cursor-plugin/plugin.json`, `.claude-plugin/plugin.json`) or docs
3. Validate Claude manifest: `claude plugin validate .`
4. Open a PR with a clear description

## Server changes

See [devutils-mcp-server contributing](https://github.com/paladini/devutils-mcp-server#-contributing).

## Reporting issues

- Plugin install / marketplace: [devutils-cursor-plugin issues](https://github.com/paladini/devutils-cursor-plugin/issues)
- Tool behavior / server: [devutils-mcp-server issues](https://github.com/paladini/devutils-mcp-server/issues)

## Code of conduct

Be respectful and constructive. This is a solo open-source project — clear, actionable reports are appreciated.
