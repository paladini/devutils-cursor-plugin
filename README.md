# DevUtils MCP

Cursor plugin for [devutils-mcp-server](https://github.com/paladini/devutils-mcp-server) — 36 local developer utilities (hash, encoding, UUID, JWT, JSON, network, text) with zero external API calls.

## Install

1. Open **Cursor Settings → Customize**
2. Search for **DevUtils MCP**
3. Click **Install**
4. Enable the **devutils** MCP server under **Customize → MCPs**

## Requirements

- [Cursor](https://cursor.com) with plugin/MCP support
- [Node.js](https://nodejs.org/) 18+

## Usage

Ask the agent to use DevUtils tools, for example:

- "Generate a UUID v4"
- "Decode this JWT"
- "Hash this string with SHA-256"

See the [devutils-mcp-server docs](https://github.com/paladini/devutils-mcp-server#readme) for the full list of 36 tools.
